# Kalman Filter Multi-Object Tracking

## 📌 Overview
This project implements multi-object tracking using Kalman Filter and background subtraction techniques.

## 🚀 Features
- Moving object detection
- Kalman Filter-based tracking
- Multi-object tracking
- Trace visualization
- Video output generation

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

## 🎯 How It Works
- Background subtraction detects moving objects
- Kalman Filter predicts object movement
- Tracks multiple objects across frames

## 📸 Output
Objects are tracked with bounding boxes and movement paths.
